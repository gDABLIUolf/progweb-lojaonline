# 🛒 VesteBem API

API REST para uma loja virtual desenvolvida com Java + Spring Boot.

O projeto tem como objetivo simular o backend de um e-commerce, permitindo gerenciamento de produtos, categorias, usuários, carrinho de compras e pedidos.

---

# 🚀 Tecnologias Utilizadas

## Backend

- Java 21
- Spring Boot
- Spring Web
- Spring Data JPA
- Spring Security
- JWT Authentication

## Banco de Dados

- PostgreSQL

## DevOps

- Docker
- Docker Compose

## Ferramentas

- Maven
- Postman
- Git/GitHub

- Cadastro de usuários

## ✅ Produtos

## ✅ Categorias

- Cadastro de categorias
- Relacionamento com produtos

- Adicionar produtos

## ✅ Pedidos
